# Рефлексия

В данном уроке мы изучим еще один алгоритм поиска, который работает не от длины массива, а от расположения определенного элемента в нем, фактически начиная от края массива. Эффективен в тех случаях, когда массив условно бесконечный или есть предположение, что элемент находится где-то в "начале".

Алгоритм экспонециального поиска:
- сначала выбираем начальное значение нашего индекса по формуле ${2^i - 2, i >= 1}$ (вначале это будет 0)
- далее исходим от сравнение с целевым элементом
	- если элемент под текущим индексом меньше целевого, тогда вычисляем следующее значение текущего индекса, затем проверяем:
		- если текущий индекс < `len(array) - 1` , тогда проверяем снова с целевым
		- иначе (текущий индекс >= `len(array) - 1`) назначаем текущему индексу значение равное `len(array) - 1`
	- если элемент под текущим зачением больше целевого, тогда мы выставляем границы диапозона для поиска и проводим в нем итерации Бинарного поиска до тех пор, пока не найдем, границы выставляем следующим образом:
		- нижняя граница - предыдущее значение текущего индекса ($(2^(i-1) - 2) + 1$)
		- верхняя граница - текущий индекс

То есть основной идеей алгоритма является нахождение верхней границы массива для поиска элемента, а затем бинарный поиск в диапазоне от верхней границы до предыдущей границы (относительно верхней).
